Cách lấy nodes khi bấm "Load Design"?

Chọn cách phù hợp nhất với workflow Figma → App

A

Multi-select từ Figma

User chọn nhiều frames trong Figma → bấm Load Design → app import tất cả nodes đang selected thành 1 Design.

Pros

  • User chủ động chọn chính xác frames nào muốn import
  • Linh hoạt — có thể chọn frames từ nhiều nơi khác nhau
  • Đơn giản để implement (Figma API đã hỗ trợ multi-selection)

Cons

  • Phải multi-select thủ công trong Figma (Ctrl/Cmd+click)
  • Dễ miss frame nếu có nhiều
B

Select parent → auto import children frames

User chọn 1 parent node (page/section) → bấm Load Design → app tự lấy tất cả direct children frames.

Pros

  • Nhanh — chỉ cần select 1 node là import cả batch
  • Phù hợp khi design đã tổ chức gọn theo section/page
  • Không sợ miss frame

Cons

  • Import hết children — có thể lấy cả frames không cần thiết
  • Phụ thuộc vào cách tổ chức file Figma
C

Kết hợp A + B (Recommended ⭐)

App kiểm tra selection: nếu user chọn nhiều nodes → import tất cả (như A). Nếu chọn 1 node có children frames → hỏi "Import children frames?" hoặc auto expand.

Pros

  • Linh hoạt nhất — hỗ trợ cả 2 workflow
  • Smart behavior: select 1 parent → auto expand, select nhiều → import trực tiếp
  • UX tự nhiên, user không cần nhớ rule

Cons

  • Phức tạp hơn để implement
  • Cần logic xác định khi nào expand vs import trực tiếp
