Create a beautiful, modern macOS-inspired Light Mode color palette applied to the desktop application shell. Our current light mode colors look unpolished and ugly, so focus purely on inventing a highly aesthetic, vibrant, and clean LIGHT MODE layout.

**DESIGN SYSTEM (REQUIRED for Light Mode):**
- Platform: Desktop (Electron), macOS-native feel
- Theme: Light Mode, incredibly clean, absolute premium aesthetics with perfect contrast.
- Background: Explore bright, airy clean backgrounds (maybe soft off-white or light frosted gray with a microscopic hint of warmth or cool tint).
- Surface: Clean white or very subtle tint for panels/cards.
- Primary Accent: A vibrant, beautiful macOS-like accent (e.g., brilliant blue, indigo, or purple, something that pops against white).
- Text Primary: Sharp, high-contrast rich dark gray (#1A1A1A or similar).
- Text Secondary: Elegant muted gray.
- Glass Panels: Light frosted glass with `backdrop-filter: blur(20px)` and very subtle white borders.
- Font: Inter for UI, JetBrains Mono for code.
- Border radius: 6px buttons/inputs, 12px cards/bento, 10px modals.
- Shadows: Soft, diffuse, layered shadows for light mode elevation (`0 4px 12px rgba(0,0,0,0.05)`).
- Compact density: 32px row heights, 36px toolbar, 28px status bar.
- NO emojis for icons — use clean SVG icons only.

**Page Structure:**
1. **Header (Toolbar):** 36px macOS toolbar with traffic lights (red/yellow/green), centered pill-shaped segmented navigation tabs, and right-aligned action icons (search, sun/moon, notifications).
2. **Left Rail:** 48px slim rail showing square-rounded project icons with an active indicator line.
3. **Sidebar (Optional):** A 240px secondary sidebar showing a list of workflow steps or items.
4. **Main Content:** A dashboard area with Bento grid cells (cards) displaying some placeholder workflow stats, demonstrating the surface and elevated surface colors cleanly.
5. **Bottom Status Bar:** A 28px footer tracking connection status (green dot), session tokens, and cost.

Focus entirely on perfecting the colors, gradients, soft shadows, and glassmorphism. Make it look like a state-of-the-art macOS light mode app!
